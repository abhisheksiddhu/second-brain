# LMIS Tech Stack - Module List

## 1. Data Layer
- **Operational Database:** Azure Cosmos DB (NoSQL API)
- **Analytics Engine:** Azure Synapse Link for Cosmos DB
- **Master Data Management:** Microsoft Purview
- **Data Ingestion:** Azure Data Factory
- **Search & Vector Store:** Azure AI Search

## 2. AI & Intelligence Layer
- **AI Orchestration:** Microsoft Semantic Kernel (C#)
- **GenAI / LLM Integration:** Azure OpenAI Service
- **Forecasting Models:** ML.NET
- **ML Ops:** Azure Machine Learning

## 3. Integration Layer
- **API Framework:** ASP.NET Core Web API
- **API Gateway:** Azure API Management
- **Event Bus:** Azure Event Hubs
- **Standardization:** OpenAPI (Swagger) & OData

## 4. Platform Layer
- **Compute:** Azure App Service (Linux)
- **Real-Time Comm:** Azure SignalR Service
- **Identity (IAM):** Azure AD B2C
- **Security & WAF:** Azure Front Door
- **Monitoring:** Application Insights

## 5. Experience Layer
- **Web Portal:** Blazor Server (.NET 8)
- **Mobile App:** .NET MAUI
- **Omnichannel:** Azure Communication Services
- **CMS:** Orchard Core
